for Loop

The for loop is perhaps the easiest JavaScript looping structure to apply. The for loop resembles a parameterized function or method and generally will include three statement parameters. The first statement is the variable declaration/initialization. The second is the test condition. The third is the variable incrementation.

while Loop

The while loop appears simpler than does the for loop, but the while loop is likely a more frequent cause of infinite loops. The while loops takes a single argument: a conditional statement. You must ensure that if the while loop is entered that the input condition at some point becomes false or that you have a break statement to escape the loop.
